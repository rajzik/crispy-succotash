# HW

## Getting started

- `pnpm install`
- `pnpm db:push`
- `pnpm dev`

### Running validations

- `pnpm test`

**Or:**

- `pnpm lint`
- `pnpm format`
- `pnpm typecheck`

## Stack

- [Next.js](https://nextjs.org)
- [Drizzle](https://orm.drizzle.team)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
